# Repo de referencia (local)

Clonar aquí el proyecto de datos y EDA binario:

```bash
git clone --depth 1 https://github.com/apostaremczak/churn-prediction.git reference/churn-prediction
```

**No versionamos** `reference/churn-prediction/` en Git (ver `.gitignore`).  
Tu pipeline de grado está en `src/` y `notebooks/` — enfoque **5 categorías de riesgo**, no clasificación binaria.

Datos KKBox → ver [`docs/DATA_SETUP.md`](../docs/DATA_SETUP.md).
